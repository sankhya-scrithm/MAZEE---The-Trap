# Binary Maze Game

A Scratch maze game created for the CS50 Scratch Project.

## Concept
This game teaches route optimization and binary logic through a maze challenge.

- The glowing maze path is actually a trap route.
- The outside edge path is the optimal route.
- Players must solve a binary question to unlock the trap path.
- Different endings appear depending on the chosen route.

## Features
- Binary puzzle system
- Multiple sprites
- Variables and conditions
- Route detection
- Trap and optimal endings
- Custom movement logic
